# 此处是项目的说明书


作业需求：
    模拟实现一个ATM + 购物商城程序
    - 额度 15000或自定义  ---> 注册功能
    - 实现购物商城，买东西加入购物车，调用信用卡接口结账 ---> 购物车功能、支付功能
    - 可以提现，手续费5% ---> 提现功能
    - 支持多账户登录 ----> 登录功能
    - 支持账户间转账 ----> 转账功能
    - 记录日常消费流水 ---> 记录流水
    - 提供还款接口 ---> 还款功能
    - ATM记录操作日志 ---> 日志功能
    - 用户认证用装饰器 ---> 登录认证



开发项目的过程:
    1.需求分析
        - 注册
        - 登录
        - 查看余额
        - 提现功能
        - 还款功能
        - 转账功能
        - 查看流水
        - 购物功能
        - 查看购物车

    2.程序架构的设计
        - 三层架构

    3.分任务开发
    4.测试
    5.上线运行
